# Antique Store Management System

Информационная система для автоматизации работы антикварного магазина.

## Технологии

- ASP.NET Core MVC
- Entity Framework Core
- MSSQL
- NUnit
- Moq
- Selenium

## Возможности

- Управление товарами
- Управление заказами
- Ролевая модель доступа
- Авторизация и аутентификация
- Формирование отчетов

## Роли

- Администратор
- Бухгалтер
- Товаровед
